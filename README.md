# Zoomi's Editor

**小説執筆から普通のメモ帳にも使える、軽量テキストエディタ**

## 概要
Zoomi's Editorは、小説やシナリオの執筆、長文のテキスト作成を快適に行うために開発されたデスクトップ用エディタです。
軽量で高速に動作し、**縦書きプレビュー、タイプライターモード、高度なアウトライン機能、そして強力なバックアップシステム**を備えています。

文字コードを自動で判別するため、古いShift-JISのファイル（青空文庫など）も文字化けすることなく一瞬で開くことができます。

## ✨ 主な特徴

* **超高速・軽量動作**
  10万文字以上の長文を貼り付けてもラグが起きず、サクサク動作します。
* **縦書きプレビュー機能（分割 / 全画面）**
  横書きで執筆しながら、下半分で縦書きのレイアウトを確認したり、全画面の縦書きビューアとして読むことに没頭したりできます。
* **文字コード自動判別（Shift-JIS / UTF-8）**
  外部から持ってきたテキストファイルも、自動で判別して文字化けせずに読み込みます。
* **タイプライターモード搭載**
  文字を入力している行が常に画面の中央にキープされるため、視線を上下に動かさず執筆に集中できます。
* **強力なアウトライン（目次）機能**
  Markdownの `#` 記号はもちろん、「第〇章」「【プロローグ】」など、自分好みの記号や正規表現を見出しとして認識させ、ダブルクリックで瞬時にジャンプできます。
* **ブックマーク（しおり）機能**
  文中に `@@` または `＠＠` と打ち込むだけで、その行が自動的にブックマークとしてメニューに登録されます。気になる箇所にすぐ戻れます。

## ⚙️ 執筆をサポートするシステム

* **充実のテーマ・スタイル設定**
  文字色や背景色を自由に変更可能（ダーク・ライト・羊皮紙などのプリセットあり）。設定はスロットに3つまで保存できます。
* **集中フェード機能**
  画面の上下をグラデーションで暗く（見えなく）し、今書いている行だけに集中できる機能です。
* **鉄壁のバックアップ**
  1分ごとの「自動保存機能」に加え、アプリ終了時には自動で「タイムスタンプ付きのバックアップファイル」を生成し、不慮のデータ消失を防ぎます。
* **よく使うフォルダ・ファイルの登録**
  メニューに特定のフォルダやテキストを登録（ピン留め）しておくことで、どこからでも一発でファイルを開けます。

## 📖 便利な使い方・TIPS

### ブックマークの活用
テキストの途中で以下のように入力すると、メニューの「ブックマーク」一覧に表示されます。
> `@@ ここを後で修正する`
> `＠＠` （文字を書かなくてもOKです）
一覧からクリックするだけで、一瞬でその行へジャンプできます。

### アウトラインのカスタム
設定画面の「アウトライン」タブから、正規表現を使って自分だけの見出しルールを作成できます。
例： `^◆.+` と設定すると、「◆」で始まる行がすべて見出しとしてアウトラインに抽出されます。

### ショートカットキーの割り当て
設定の「キー」タブから、各種機能（縦書きビューの切り替え、文字数カウントの表示/非表示など）に好きなショートカットキーを割り当てることができます。

## 📥 インストール・起動方法

1. 配布ファイル（zipなど）をダウンロードして解凍します。
2. フォルダ内にある実行ファイル（`.exe` など）をダブルクリックすると起動します。
※インストールの必要はありません。USBメモリなどに入れて持ち運ぶことも可能です。
